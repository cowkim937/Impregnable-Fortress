# Impregnable Fortress

브라우저에서 실행되는 성벽 방어/채굴 성장 게임입니다. 성벽을 지키면서 자원을 채굴하고, HRC 소재를 업그레이드하며, 상점 아이템과 자동 유닛으로 몰려오는 적과 부락을 막는 방식입니다.

## 프로젝트 구조

```text
.
├── index.html          # 게임 화면과 정적 리소스 연결
├── public/css/
│   └── style.css       # 게임 UI 스타일
├── public/sounds/      # CC0 8비트/레트로 효과음 MP3
├── src/
│   ├── main.js         # 브라우저 실행 진입점
│   ├── game.js         # 게임 루프, 전투, UI, 렌더링 로직
│   └── config.js       # 난이도, 아이템, 병종, 건물, HRC 데이터
├── server.py           # 로컬 개발용 정적 파일 서버
├── Dockerfile          # 게임 서버 이미지
├── Dockerfile.sql      # SQLite 볼륨 초기화 이미지
└── docker-compose.yml  # 로컬 배포 구성
```

## 주요 시스템

| 시스템 | 설명 |
| --- | --- |
| 성벽 방어 | 원형 성벽 세그먼트가 피해를 받으며, 수리 아이템과 성장으로 버팁니다. |
| 채굴 | 클릭 채굴과 채굴로봇으로 나무, 돌, 원석, 금화를 획득합니다. 모든 난이도 채굴량은 상향 조정되어 있습니다. |
| HRC 업그레이드 | HRC 1~99 소재 단계가 있으며 40단계 이후 비용은 난이도별 `2*n*log(n)` 기반 로그 곡선을 사용합니다. |
| 병력/무기 | 돌, 청동 도끼, 철단검, 총, 번개창, 중력파가 HRC 단계에 따라 개방됩니다. |
| 마법 효과 | 화염, 물, 전기, 소멸, 대지균열 등은 조건에 따라 랜덤으로 1개 또는 2개만 발동합니다. 공격/효과에는 전투 텍스트가 표시되지 않습니다. |
| 부락 | 10초 동안 투명도와 능력치가 20%에서 100%로 성장합니다. 성장 중에도 피해를 줄 수 있어 사전 제거가 가능하며, 완성 후 몬스터를 생성합니다. |
| 채굴로봇 | 최대 20기까지 구매합니다. 로봇은 성과 채굴장을 왕복하며 채굴장 도착 후 2초 채취하고, 성에 복귀하면 클릭 대비 4배 자원을 지급합니다. |
| 업그레이드 상인 | 최대 10기까지 구매합니다. 2~3초마다 시장에서 자원을 얻고, 상인 1기당 시장 획득률이 10% 증가합니다. |
| 사운드 | Kenney CC0 오디오 팩에서 추출/변환한 8비트/레트로 효과음을 `SoundManager`로 관리합니다. |
| 랭킹 | 게임 종료 시 브라우저 `localStorage`에 기록하고 상위 20개를 조회합니다. 서버 API 없이 GitHub Pages에서도 동작합니다. |

| 쉬움 | 적 약화와 스폰 감소, 부락 90초마다 생성, 부락 HP 0.7배 |
| 보통 | 기준 난이도, 부락 60초마다 생성, 부락 HP 1배 |
| 어려움 | 적 전투력 상승, 부락 50초마다 생성, 부락 HP 2배 |
| 극한 | 고난이도 기준 비용과 적 전투력, 부락 30초마다 생성, 부락 HP 5배 |
| They are billions | 속도/스폰/부락 강화, 성벽 체력 감소, 부락 5초마다 생성, 부락 HP 8배 |

## 상점 아이템

| 아이템 | 효과 |
| --- | --- |
| 성벽 수리 🔨 | 선택한 성벽 세그먼트 HP를 50% 회복합니다. |
| 지뢰 💣 | 선택 위치 주변 적에게 큰 피해를 줍니다. |
| 탄도 미사일 🚀 | 넓은 범위의 적과 부락을 타격합니다. |
| 핵무기 ☢️ | 탄도 미사일 7배 범위에 `1B * HRC 레벨 + 최대 체력 80%` 피해를 주며, 60초 방사능 잔존 효과를 남깁니다. 최대 2기 보유 가능하고 구매할 때마다 가격이 1T씩 증가합니다. |
| 야근 🌝 | 60초 동안 채굴량이 2배 증가합니다. |
| 위성 레이저 🛰️ | 추적 레이저가 적을 집중 타격합니다. |
| 썬더러 🌩️ | 일정 시간 동안 보라/흰색 수직 낙뢰를 호출합니다. |
| 윈디 🌬️ | 전체 적에게 바람 피해와 감속을 적용하고, 파란 줄 이펙트가 각기 다른 투명도로 움직입니다. |

## 업그레이드와 비용

- HRC 업그레이드는 99단계까지 진행됩니다.
- 40단계 이후 비용은 로그 곡선을 사용합니다.
- 극한 난이도 기준으로 80단계 약 1T, 90단계 약 10T, 99단계 약 99T 수준입니다.
- 어려움은 극한의 1/3, 보통은 1/6, 쉬움은 1/15, 매우 쉬움은 1/25 비용을 적용합니다.
- 채굴로봇 1~5기와 업그레이드 상인 1~5기는 기존 비용의 1/50로 저렴하며, 이후 가격은 기존 곡선을 따릅니다.
- 채굴로봇 20기 가격 목표는 보통 난이도 기준 10B입니다.
- 업그레이드 상인 10기 가격 목표는 보통 난이도 기준 5B입니다.
- 무기 진화는 1단계 30초부터 시작해 다음 단계마다 시간이 2배씩 증가합니다.

## 사운드 에셋

`download_sounds.sh`는 Kenney의 CC0 오디오 팩을 직접 다운로드하고, 선택한 OGG 파일을 MP3로 변환해 `public/sounds/`에 저장합니다.

```bash
./download_sounds.sh
```

필요 도구: `curl`, `unzip`, `ffmpeg`

## 관리자 치트

개발/검증용으로 닉네임 또는 콘솔에서 다음 키를 사용할 수 있습니다.

```js
IamAdmin("cowkim")
```

활성화되면 모든 자원과 체력이 무한대로 표시되며, 기록 이름은 `-!-!-ADMIN-!-!-`으로 저장됩니다.

## 로컬 실행

Python 표준 라이브러리만으로 실행할 수 있습니다.

```bash
python3 server.py
```

브라우저에서 `http://127.0.0.1:8000/`을 엽니다.

## Docker 실행

```bash
docker compose up -d --build
```

브라우저에서 `http://127.0.0.1:12321/`을 엽니다.

중지하려면 다음을 실행합니다.

```bash
docker compose down
```

## 검증

자바스크립트와 서버 문법 확인:

```bash
node --check src/main.js
node --check src/config.js
node --check src/game.js
python3 -m py_compile server.py
```

브라우저 동작은 Playwright로 로컬 또는 Docker 배포 주소에 접속해 시작 화면, 상점/인벤토리, 관리자 키, 핵무기/로봇/상인 UI를 확인합니다.
